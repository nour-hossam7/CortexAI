# Datasets Guide

This folder contains the dataset structure for **CortexAI**.

The project uses two raw datasets:

- **BraTS2020 Training Data (Kaggle)** → for MRI image analysis / segmentation
- **TextBraTS** → for radiology-style text reports / multimodal text support

---

## Dataset Structure

```bash
datasets/
├── raw/
│   ├── brats2020/
│   └── textbrats/
│
├── processed/
│   ├── cv/
│   ├── nlp/
│   └── fusion/
│
└── sample_data/

Raw Datasets
1) BraTS2020

This folder stores the raw MRI / segmentation data used by the Computer Vision module.

Expected location:

datasets/raw/brats2020/

BraTS files should remain inside this folder and keep their original patient-level structure.

2) TextBraTS

This folder stores the text-based reports / text features used by the NLP and Fusion modules.

Expected location:

datasets/raw/textbrats/

TextBraTS files should remain inside this folder.

Processed Data Folders

Processed outputs generated by preprocessing scripts should be saved inside:

CV processed outputs
datasets/processed/cv/
NLP processed outputs
datasets/processed/nlp/
Fusion-ready outputs
datasets/processed/fusion/
Team Setup Instructions

After cloning the repository, each team member should:

1) Install dependencies
pip install -r requirements.txt
2) Download / place the datasets
Put BraTS2020 inside:
datasets/raw/brats2020/
Put TextBraTS inside:
datasets/raw/textbrats/
3) Run the dataset setup check

From the project root, run:

python -m src.utils.setup_data

This script will:

create missing required folders if needed
verify whether BraTS2020 exists
verify whether TextBraTS exists
confirm that processed directories are ready
Notes
Raw medical datasets are not uploaded to GitHub because of size limitations.
All team members should keep the same local folder structure.
Processed outputs should not be committed unless explicitly needed for experiments or demos.
Use sample_data/ only for tiny demo examples if needed.

Natural Language Processing (NLP)
Input dataset: TextBraTS
Raw path: datasets/raw/textbrats/
Processed output path: datasets/processed/nlp/
Multimodal Fusion
Inputs: processed CV features + processed NLP features
Output path: datasets/processed/fusion/
Recommended Team Workflow
CV teammate
Read BraTS2020 from datasets/raw/brats2020/
Preprocess MRI data
Save outputs inside datasets/processed/cv/
NLP teammate
Read TextBraTS from datasets/raw/textbrats/
Preprocess / clean / embed text reports
Save outputs inside datasets/processed/nlp/
Fusion teammate
Load processed outputs from:
datasets/processed/cv/
datasets/processed/nlp/
Build multimodal fusion inputs
Save final fusion-ready files inside datasets/processed/fusion/
Important Rule for the Team

Please do not change folder names or paths.

All modules in CortexAI are expected to use the following standard paths:

datasets/raw/brats2020/
datasets/raw/textbrats/
datasets/processed/cv/
datasets/processed/nlp/
datasets/processed/fusion/

Keeping the same structure for everyone ensures that the code runs consistently across all team members' machines.
